WPPSniffStorage
===============

WPP Sniff storage.

Credentials are stored at pair.txt.dist rename it to pair.txt after merge on 1st setup.

Default credentials: root / toor

tolower(username):sha1(salt + upper(user) + ":" + upper(pass) + salt) to generate new credentials.
